## 実行時間制限とOrder
Pythonの場合
- O(1e5)/sec: 余裕を持って間に合う  
- O(1e6)/sec: おそらく間に合う  
- O(1e7)/sec: 非常にシンプルな処理でない限り難しい

- list.index(value): index取得
